# 鹈鹕游戏官网 - 快速部署指南

## 一键部署命令

### 方式一：Vercel CLI（推荐）

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录 Vercel
vercel login

# 3. 进入官网目录
cd 鹈鹕游戏/官网

# 4. 首次部署（预览）
vercel

# 5. 生产环境部署
vercel --prod
```

### 方式二：拖拽部署（最简单）

1. 访问 [vercel.com/new](https://vercel.com/new)
2. 直接将 `鹈鹕游戏/官网` 文件夹拖入页面
3. 等待 30 秒，网站自动上线

### 方式三：Netlify（无需命令行）

1. 访问 [app.netlify.com/drop](https://app.netlify.com/drop)
2. 直接将 `鹈鹕游戏/官网` 文件夹拖入页面
3. 等待部署完成，获得免费域名

## 部署后配置

### 配置域名（可选）

部署完成后，平台会提供免费域名

如需自定义域名：
1. 进入平台 Dashboard → 项目 → Settings → Domains
2. 添加你的域名（如 `pelican-games.com`）
3. 按提示配置 DNS 记录

## 常见问题

**Q: 部署后网站显示空白？**
A: 检查浏览器控制台是否有错误，可能是文件路径问题。

**Q: 如何更新网站内容？**
A: 修改文件后重新执行 `vercel --prod` 或拖拽更新即可。
