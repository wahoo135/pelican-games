## 网站预览

### 本地预览方式

1. **直接打开**：双击 `index.html` 文件在浏览器中打开
2. **使用本地服务器**：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (npx)
   npx serve .
   ```
   然后访问 `http://localhost:8000`
