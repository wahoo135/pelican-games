## 🚀 部署步骤

### 方式一：Vercel CLI 部署（推荐）

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录 Vercel
vercel login

# 3. 进入官网目录
cd 鹈鹕游戏/官网

# 4. 首次部署
vercel

# 5. 生产环境部署
vercel --prod
```

### 方式二：Netlify 部署（最简单）

1. 访问 [app.netlify.com/drop](https://app.netlify.com/drop)
2. 直接拖拽整个 `官网` 文件夹到页面
3. 等待部署完成，获得免费域名

### 方式三：Git 部署

1. 将 `鹈鹕游戏/官网/` 目录推送到 GitHub/GitLab 仓库
2. 访问 [vercel.com](https://vercel.com)
3. 点击 "Import Project"
4. 选择你的仓库
5. 配置项目后点击 "Deploy"
